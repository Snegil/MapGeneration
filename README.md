# MapGeneration

Procedural map generation experiment written in C# and Unity.

## Description
MapGeneration uses the **Drunkard's Walk Algorithm** to procedurally generate 2D maps.

## Features
- Procedural terrain generation
- Saving generated maps to a file
- Loading a saved map
  
## Future Features
Nothing at this point in time.

## How to run
1. Download `.zip` from the **[releases](https://github.com/Snegil/MapGeneration/releases)**
2. Extract the files
3. Run `.exe` file
