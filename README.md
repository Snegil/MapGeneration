# MapGeneration
Procedural map generation experiment written in C# and Unity.

## Description
MapGeneration uses the **Drunkard's Walk Algorithm** to procedurally generate 2D maps.

## Features
- Procedural terrain generation

## Future Features
- Saving generated maps to a file
- Loading a saved map

## How to run
1. Download `.zip` from **[releases](https://github.com/Snegil/MapGeneration/releases)**
2. Extract the files
3. Run `.exe` file
